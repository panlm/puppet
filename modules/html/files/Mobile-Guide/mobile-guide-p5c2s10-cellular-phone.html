16.10. Connection to Cellular Phone

AFAIK there are two methods to connect a cellular phone to a laptop: via the infrared port (IrDA®) or via the serial port. See the Linux/IrDA® project for the current status of IrDA® connections. As far as I know only the Ericsson SH888, the Nokia 8110 and the Siemens S25 provide infrared support.
