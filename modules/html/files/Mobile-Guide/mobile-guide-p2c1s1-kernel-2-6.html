14.2. Kernel 2.6

14.2.1. PCMCIA

PCMCIAutils contains hotplug scripts and initialization tools necessary to allow the PCMCIA subsystem to behave (almost) as every other hotpluggable bus system (e.g. USB, IEEE1394). Please note that the kernel support for this new feature is only present since 2.6.13-rc1.
