12.27. GPRS

GPRS is a General Packet Radio Service, an add-on to GSM and TDMA cellular telephone standards used all over the world. It allows (almost) always-on Internet connections using GSM (or TDMA) telephones. It makes mobile internet usage on laptops fairly inexpensive. The GPRS-HOWTO is written by Esa Turtianen etu_AT_dna.fi and Jari Arkko Jari_AT_arkko.com
