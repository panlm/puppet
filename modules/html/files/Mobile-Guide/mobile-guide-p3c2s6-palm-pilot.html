7.4. PALM-Pilot

7.4.1. Resources

  1. 3COM is the manufacturer of the PALM-Pilot.

  2. PalmOS-HOWTO (former Pilot-HOWTO) by David H. Silber.

  3. PilotLink and XCoPilot PilotLink is an utility that performs data transfers from 3com PalmPilot handheld computers to your Linux machine. XCoPilot is an emulator of the PalmPilot operating system that runs under Linux.

  4. ucLinux

  5. PalmVNC is an implementation of the Virtual Network Client architecture that will allow you to use a Linux or other UNIX machine to put up a (tiny) X Window on a 3COM PalmPilot.

  6. Survey of Linux and BSD Applications for the Palm
